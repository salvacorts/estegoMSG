#EstegoMSG

Oculta y Revela mensajes en imágenes PGM o PPM

Proyecto desarrollado junto a [Ruben58](https://github.com/ruben58)

###Uso:

- *Ocultar:* `$ ./bin/ocultar`
- *Revelar:* `$ ./bin/revelar`

###Compilación:

```bash
$ make
$ make clean
```

#Documentación (doxygen)

`$ make documentacion`
